# Case Studies Overview

This document summarizes Raaj’s key portfolio projects in a recruiter-friendly format.

---

## End-to-End Automated Data Pipeline & Analytics Platform

### Problem
Manual data ingestion and reporting workflows caused delays, inconsistency, and low trust in analytics outputs.

### Approach
- Designed automation-first ETL pipelines using Python and SQL
- Implemented data validation and schema consistency checks
- Modeled analytics-ready datasets for BI tools
- Optimized pipelines for reliability and scalability

### Impact
- Reduced manual reporting effort
- Improved data reliability and trust
- Enabled faster turnaround for analytics requests

### Stack
Python, SQL, Airflow, PostgreSQL, AWS S3, Data Modeling, BI Tools

---

## AI-Adaptive Web Scraping Framework

### Problem
Scraper scripts frequently broke due to changing website structures, requiring constant manual fixes.

### Approach
- Built a Python-based framework that uses LLMs (DeepSeek) to generate scraper logic
- Automated script generation based on target website structure
- Standardized scraped outputs into structured datasets and schemas

### Impact
- Reduced time and effort to build new scrapers
- Improved resilience to DOM and layout changes
- Accelerated ingestion of external data sources

### Stack
Python, LLMs (DeepSeek), Web Scraping, Automation, APIs, Data Modeling

## Contract Analytics Platform

### Problem
Contract and vendor data was fragmented and difficult to analyze consistently.

### Approach
- Built structured datasets from contract data
- Defined reusable KPIs and metrics
- Developed interactive dashboards for comparison and trend analysis

### Impact
- Enabled faster vendor evaluation
- Improved contract performance visibility
- Supported data-driven decision-making

### Stack
SQL, Power BI, Tableau, Data Modeling, KPI Design

## E-commerce Delivery Risk Insights

### Problem
Delivery delays lacked clear root-cause visibility and risk quantification.

### Approach
- Built analytics models to analyze delay drivers
- Created feature pipelines for risk factors
- Visualized trends and geographic hotspots

### Impact
- Identified high-risk delivery patterns
- Supported proactive decision-making
- Improved operational insights

### Stack
Python, SQL, EDA, Dashboards, Analytics
